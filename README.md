## 📖 Overview

SipraEdu Website is a modern educational platform developed by **Sipra Corporation** to showcase professional IT training programs. The website provides an engaging user experience with responsive design, training information, contact forms, and online registration capabilities.

---

## ✨ Features

- Modern Responsive Design
- Professional Landing Page
- Interactive Hero Section
- About SipraEdu
- Training Programs
- Student Registration
- Contact Form
- Social Media Integration
- Mobile Friendly
- SEO Optimized
- Fast Loading
- Professional UI/UX

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/SipraCorporation/sipraedu-website.git
```

### Backend

- Open the ASP.NET Core solution.
- Restore NuGet packages.
- Configure the connection string.
- Run the application.

### Frontend

```bash
npm install
npm start
```

### Database

- Create a Microsoft SQL Server database.
- Import the SQL scripts.
- Update the connection string.
- Launch the application.
